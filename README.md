# [과제_5] 05. 리액트를 이용한 CRUD 앱

## 📝 개요

React와 TypeScript를 이용하여 간단한 CRUD 기능을 가진 To-Do 리스트 앱을 구현한 과제입니다.  
사용자는 할 일을 추가, 수정, 삭제하고 완료 여부를 체크할 수 있습니다.

---

## 🔹 기능

- 할 일 추가
- 할 일 수정 (Save / Cancel)
- 할 일 삭제
- 할 일 완료 체크 (체크 시 글자에 취소선 적용)

---

## 💻 기술 스택

- React (Functional Component, Hooks)
- TypeScript
- CSS
